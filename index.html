Rules

Cryptoquote uses a substitution cipher that changes each letter of the alphabet to a different letter or other symbol.

Each letter gets a point value that the player pays to reveal the letter.

Players start with 500 points. When they select a letter, they pay a set amount for the letter choice multiplied by the number of times the letter is used in the quotation.

If the player solves the cryptoquote, the value of all the remaining letters in the quotation is credited to the score. If the player spends all the points before solving the cryptoquote, it is a loss.

A bonus question will be offered after a successful constant that adds 100 points to the score.
